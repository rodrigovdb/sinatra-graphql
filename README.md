# Sinatra with GraphQL

## Requirements

* Ruby 2.5.0
* Sqlite3

After have your workcopy cloned into your machine, run
```
$ bundle install
```

## Running locally

If you want use shotgun, run
```
$ bundle exec shotgun -p 9292
```

If you want run it with rack, use
```
$ bundle exec rackup -o 0.0.0.0
```
